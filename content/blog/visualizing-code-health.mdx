# Visualizing Code Health: Introducing Project Pulse

Project Pulse is a new feature in the GitLegend dashboard that gives you a high-level overview of your project's health. It visualizes key metrics like code churn, commit frequency, and contributor activity, allowing you to quickly identify trends and potential issues.
